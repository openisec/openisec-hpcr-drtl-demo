# Contributing to HPCR-DRTL

Thank you for your interest in HPCR-DRTL!

## Project Policy: Open source, maintainer-led

This repository is published primarily as a reference implementation and public demo of the HPCR-DRTL framework.

At this early stage, the core implementation is maintained by the Openisec maintainer.
We are not actively accepting external code contributions or pull requests at this stage.

## How You Can Contribute

We genuinely welcome the following through GitHub Issues:

- 🐛 **Bug reports** — Something not working as expected
- 💡 **Feature suggestions** — Ideas for improvement
- 📋 **Use case feedback** — How you are using or would use HPCR-DRTL
- 🔍 **Security concerns** — Please report via email to admin@openisec.com
- 📝 **Documentation improvements** — Unclear or missing documentation

## Forking

You are welcome to fork this repository and experiment with your own version
under the terms of the Apache License 2.0.

## Contact

For questions, demo access requests, or business inquiries:

## Code of Conduct

Please be respectful and constructive in all interactions.
